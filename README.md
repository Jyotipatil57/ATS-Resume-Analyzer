# AI ATS Resume Analyzer

## Overview
The AI ATS Resume Analyzer is a web application built using Python and Streamlit. It analyzes resumes against job descriptions and calculates an ATS compatibility score.

## Features
- Upload PDF or DOCX resumes
- Extract resume text
- Compare resumes with job descriptions
- Calculate ATS score
- Detect skills
- Show missing skills

## Technologies Used
- Python
- Streamlit
- pandas
- pdfplumber
- python-docx
- scikit-learn

## How to Run

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
